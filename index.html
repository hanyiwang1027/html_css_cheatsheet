Semantic HTML Reference

Semantic HTML Tags

Tag Name Description
<header> header section represents introductory content, typically a group of introductory or navigational aids
<nav> navigation section encapsulates the page’s navigational links; often placed inside the <header> or <footer>
<main> main section represents the dominant content of the <body> of a document
<footer> footer section includes the page’s footer content at the bottom of the <body>
<video> video embed element embeds a media player which supports video playback into the document
<audio> audio embed element embeds sound content in documents; may contain one or more audio sources
<embed> embed external content element implements any type of media
<figure> figure with optional caption element encapsulates media such as an image, diagram, or code snippet
<figcaption> figure caption describes the media encapsulated within the <figure> element
<section> generic section element defines elements in a document, such as chapters, headings, or any other area of the document with the same theme
<article> article conent element holds content that makes sense on its own such as articles, blogs, and comments
<aside> aside element marks additional information that can enhance another element but isn’t required in order to understand the main content

Semantic HTML Attributes

Attribute Name Description
controls controls the browser will offer controls to allow the user to control video playback, including volume, seeking, and pause/resume playback
loop loop the browser will automatically seek back to the start upon reaching the end of the video
muted muted the audio will be initially silenced
src source the URL of the video to embed
height height the height of the video's display area, in CSS pixels
width width the width of the video's display area, in CSS pixels
type type The MIME type to use to select the plug-in to instantiate
alt alternate provides fallback text to display when the image is not loaded; a property to support accessibility
